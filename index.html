Cómo conectar Strava

  1. Ve a strava.com/settings/api →
  2. Inicia sesión si hace falta
  3. Rellena el formulario:
    · Application Name: cualquier nombre (ej. "TrailCoach")
    · Category: Training
    · Website: puedes poner cualquier URL
    · Authorization Callback Domain:
  4. Haz clic en Save o Update
  5. Copia el Client ID (número) y el Client Secret y pégalos en Ajustes
